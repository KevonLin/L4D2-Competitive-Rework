# **CLVPlus**

* mix插件
* sourcebans套件
* lilac反作弊
* sm_hp回血功能。
* all4dead2控制。
* sm_mpg改旁观。
* 显示连接、城市、IP等信息。
* 禁止投票返回大厅
* 旁观作弊
* 改旁观后自动关闭匹配
* OP免踢
* rygive套装
* 服务器存储聊天记录
* 展示正在语音的人
* 自动换图
* 复活传送插件
* 服务器自动重启
